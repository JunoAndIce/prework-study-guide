HTML


Types of Elements:


These are the main elements that control the content on a webpage.

The most commonly used types are:


  • Text Elements
  • Image Elements
  • Header Elements
  • List Elements

Alongside these element types there are also element types that control and maintain the page's formatting.


  • Meta-Related Elements
  • Content Division Elements
  • Semantic Elements

Two main elements needed for HTML webpage coding are Meta-related and those are the '<head>' and '<body>' elements.


  • Head: The '<head>' element contains main information on the wepbage, and includes the title. Information typed in this element is not visable on a webpage.
  • Body: The '<body>' element contains the visible content on the wepbage such as images and relevant text. Information that is necessary for the user is typed between this element.

Within the '<head>' element are also more elements that help give relevant information that is important for the webpage to function.


  • Meta: The '<meta>' element defines the metadata of a document. The meta element is used to detail the character set, page descriptions, viewport, author, and any important keywords of the webpage.
  • Title: The '<title>' element defines the title of the webpage. The title element is useful for search engines, and other tools in displaying your webpages name.
  • Link: The '<link>' element creates connections to external files. These files could be used for many functions such as CSS and image displaying, and much more.

CSS


The CSS controls the style and look of a wepbage.

CSS can be written on the same document as the HTML code, but is generally written on a seperate document, and linked to the HTML code.

A CSS document can directly interact with elements in the HTML document. For example, you can directly reference all <h1>'s on the HTML document and give them different colors, or text size.


Some General CSS notes:

  • You can effect the elements of anything from color, height, width, font-size, margin, and padding.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Color effects the text color for a specific block. Background-color is the color of the element's background.

Git


Git is a free open source version control system useful for maintaining projects. It offers the ability to rollback to older projects if problems arise in new changes, and allows cooperative work on a codebase without overwriting your peer's work and allows all users to discuss and agree on a new change that can then be pushed into the main project.

Git is very useful for all codebase projects and almost necessary for organizing work across a large project.

Git allows you to work on your project locally and keep the original branch intact. Then you can push any changes in a side branch back into the main branch safely.


Git terms:


  • pwd: Print working directory
  • ls: Show folders in current directory
  • cd: Change directory
  • cd..: Move back one directory
  • mkdir: Make a new directory
  • touch: create files
  • start: Opens target file

General Git commands:


  • git status: checks what current branch we're in.
  • git checkout -b: Creates a new branch and changes to it
  • git add -A: Add changes to branch
  • git commit -m: Commit changes to branch
  • git pull origin main: Pull from the main branch
  • git push origin: Push to the main branch

  • -A: All changes
  • -b: New Branch
  • -m: New Message
  • main: main branch
  • pull: recieve branch's modifications into local branch
  • push: add new changes to an existing branch
  • origin: Source (Main branch/feature branch)

JavaScript


JavaScript gives interactivity to a website. JavaScript allows clickable buttons, links and much more to function, which give life to a webpage.

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Variables can be anything, as JavaScript is a dynamic language with dynamic types. Javascript variables are not directly associated with specific types.
  • JavaScript allows implicit type conversions automatically which means that adding a string to a integer will coerce the int into a string to allow them to be properly used together without throwing errors.

Arrays are a useful way of holding multiple different variables inside of just one variable.

Variables can be delcared in an array form by using brackets [ ], and inputting the relevant information inside the brackets.

For example: var topics = ['HTML', 'GIT', 'CSS', 'JavaScript'] will store four string variables inside of the topics variable.

To call a variable in an array, you type the variable name and it's index number in brackets. The index number is the location of the variable in the array.

Calling 'HTML' will require the user to type, topics[0] where topics is the array where 'HTML' is located in, and 0 is the index. The first slot in an array is always 0.


For-Loops require three parts:

  • The Starting Point: By delcaring a variable Ex. (var x = 0) The for loop now has a starting point at 0 using x as it's variable.
  • The Condition Statement: The condition statement tells the for loop a rule that it will uphold in order to keep the loop going, and as long as that rule remains true, the loop will continue. Ex. (x < topics.length). This information tells us that as long as X is less than the length of the topics array, it will continue to loop.
  • The Variable Increment: Finally, you will increment the starting variable. This depends on how your array is handled, but usually you will increment the value by 1 using (x++). Now when the loop completes once, it will go back through but now with +1 added to x.

This usually contiues until the condition statement is no longer true, which is when the for-loop will stop.
